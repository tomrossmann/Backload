Welcome.

Backload is a professional, full featured file upload controller and handler (server side) for ASP.NET MVC 4, MVC 5, Web API and standard ASP.NET Web Forms. While Backload out of the box handles the client side jQuery File Upload Plugin from blueimp, PlUpload from Moxiecode and Fine Uploader from Widen Enterprises, it can be easily customized to work with any client side plugin - or standalone.

General Information

Background

Backload is a feature rich, fully customizable server side file upload component for ASP.NET (MVC, WebAPI, Web Forms) with roots in a project management and documentation suite for the aero craft industry. It has proved its scalability in customer projects with 100,000 and more uploads per day.

Preserve your investment

Backload supports the most commonly used client side file upload plugins out of the box, and it can be customized to support other plugins very easily (see examples). Using Backload, you are not bound to a client plugin as you can switch to a different plugin with a single setting in the configuration. Actively supported client side plugins are the JQuery FIle Upload Plugin from bluimp, PlUploaf from Moxiecode and Fine Uploader from Widen Enterprises. Note: Client side plugins are not absolutely necessary as you can easily target Backload with your own custom Ajax call.

Rich feature set

Backload has a very rich set of features. It supports complex storage structures (without a single line of code), automatic thumbnail creation, image processing, support for several client side file upload plugins, file bundeling and more. All features can be customized declaratively (config file) or in your code.

Scalability

From the first line of code, Backload has been explicitly developed with scalability in mind. It is used by some customers with huge amount of uploaded files (100k and more per day) and very large file sizes up to 1GB).

Start immediately

Integrating any kind of JSON protocols of client side plugins can be a tedious task, and it is a pain to switch to a different plugin if necessary in later development phase. With one of Backloads NuGet packages you can start immediately. The Core Package installs only the component itself and makes the necessary settings in the Web.config. The Productivity Package additionally installs the client side JQuery File Upload Plugin ready for use and a small demo website. The Demo Package installs a full demo in your project.

Actively developed

This project is actively developed by senior software developers and architects and it is backed by leading software companies. This guaranties continuously development and the integration of new emerging technologies, trends or programming pattern.
